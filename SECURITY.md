---

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

### Contact

- **Developer:** Yash Rajeshwar Lamdade
- **Email:** shivajijuniorcollege@gmail.com

Please include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Screenshots or proof (if available).
- Suggested fixes (optional).

---

## Response Policy

- Reports will be reviewed as soon as possible.
- Valid security issues will be fixed in future updates.
- We appreciate responsible disclosure and will work to resolve confirmed vulnerabilities promptly.

---

## Scope

This policy applies to:
- Website source code
- HTML, CSS, and JavaScript files
- Images and downloadable resources
- Future updates to this project
